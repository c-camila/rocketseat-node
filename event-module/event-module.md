# Event Module

É um mecanismo para

- [] Disparar eventos.
- [] Ouvir eventos.
- [] Fazer alguma ação quando ouvir o evento.
- [] É a base para muitos módulos, como: http, stream, file system, etc...

# O que aprendi

- [] Utilizar os eventos.
- [] Disparar eventos.
- [] Ouvir eventos.
- [] Executar ações para determinados eventos
- [] Entender como ele é a base para os outros módulos.
